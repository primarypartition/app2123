# TypeScript

## Setup

```
> npm install typescript lite-server --save-dev
```

## Use

```
> tsc file.ts
> tsc file.ts --watch
> npm start
```

## Config

```
> tsc --init
> tsc
> tsc -w
```

## Webpack

```
> webpack
> webpack-dev-server
> npm run start
> npm run build
> webpack --config prod.js
```

## React

```
> npx create-react-app . --typescript
> create-react-app . --typescript
```

## NodeJS

```
> tsc app.ts
> node app.js
> node app.ts

> npm init
> tsc --init
> tsc -w
> nodemon dist/app.js
```
